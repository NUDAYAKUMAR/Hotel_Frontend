# 🍽️ Hotel Frontend Website

## 📌 Description

This project is a responsive frontend website developed for my hotel to showcase menu items, services, and contact details.

It provides a clean and user-friendly interface for customers to explore food options and get information about the hotel.

---

## 🚀 Features

* 🏠 Home page with hotel overview
* 📋 Menu page with food items
* 📞 Contact page
* ℹ️ About section
* 📱 Fully responsive design
* 🎨 Attractive UI with images

---

## 🛠️ Tech Stack

* HTML
* CSS
* JavaScript

---
